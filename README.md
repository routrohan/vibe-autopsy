# HUMBLE ME

I wanted to learn Google ADK and was feeling a little too good about myself, so I built this to humble me. 

You put in your name and some context, and a bunch of AI run after your digital footprint to tear your professional persona to shreds. 

### Tech Stack
* **Frontend:** React + Vite + Tailwind (because I like green text on black backgrounds)
* **Backend:** Flask + Python
* **Orchestration:** Google ADK (Agent Development Kit)
* **Brain:** llama-3.1-8b-instant (via Groq because I ran out of gemini credits, gemini boooo)
* **Scout:** Tavily Search API (don't misuse my free credits)

*Stay humble.*
